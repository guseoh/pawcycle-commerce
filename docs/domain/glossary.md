# 도메인 용어집(Domain Glossary)

이 문서는 PawCycle Commerce에서 사용하는 공통 도메인 언어를 정의한다. 승인된 작업이나 제안된 작업에서 실제로 필요한 용어만 추가한다.

## 용어

### 고객(Customer)

일반 구매 또는 정기배송 구독으로 반려동물 소모품을 구매하는 사용자다.

### 반려동물 소모품(Pet Supply)

사료, 간식, 고양이 모래, 배변 패드, 탈취제, 샴푸 및 위생용품처럼 반복 구매가 발생하는 상품이다.

### 구독(Subscription)

하나 이상의 상품을 반복 배송받는 약정이다. 구독 가능 여부, 배송 주기, 건너뛰기, 일시정지, 재개, 해지, 결제 재시도, 재고 부족 정책은 구현 전에 승인되어야 한다.

### 배송 주기(Delivery Cycle)

정기배송 시점을 계산하는 간격이다. 지원할 주기 옵션은 아직 승인되지 않았다.

### 정기 주문(Recurring Order)

구독에서 특정 회차 배송을 위해 생성되는 주문이다. 주문 스냅숏(Order Snapshot) 정책은 도메인 설계 단계에서 승인되어야 한다.

### Product Decision

Product Owner인 사용자가 결정해야 하는 비즈니스 또는 제품 정책 결정이다.

### Technical Decision

아키텍처, 계약, 의존성, 장기 유지보수성에 영향을 주며 Tech Lead인 사용자가 결정해야 하는 기술 선택이다.

## 용어 추가 규칙

새 용어를 추가할 때는 다음을 포함한다.

- 용어명
- 짧은 정의
- 승인 출처 또는 작업 ID
- 남아 있는 정책 질문
