# 제품 문서(Product Documents)

이 디렉터리는 PawCycle Commerce의 제품 기획 산출물을 보관한다.

사용 범위는 다음과 같다.

- 제품 브리프(Product Brief)와 PRD(Product Requirements Document)
- 사용자 스토리(User Story)
- 기능 범위와 제외 범위
- 정상 흐름(Happy Path)과 예외 흐름(Exception Path)
- 제품 관점의 비즈니스 규칙(Business Rule)
- 인수 조건(Acceptance Criteria)
- Product Decision 목록

기술 구현 세부 사항, API 내부 설계, 데이터베이스 스키마(Database Schema), 애플리케이션 아키텍처(Architecture)는 이 디렉터리에서 확정하지 않는다.
