# 서비스 비전

## 문서 상태

- 작업 ID: `PS-001`
- 역할: Product Planner
- 상태: PS-001 기준 제품 결정안
- 기준 도메인: 반려견/반려묘 소모품 커머스, 단건 구매, 반복 구매, 정기 구독
- 범위 제한: 이 문서는 사용자 문제, 가치 제안, MVP 경계만 다룬다. 도메인 모델, 데이터 구조, API, 상태 전이는 DOMAIN-001 이후 작업에서 다룬다.

## 한 문장 비전

PawCycle Commerce는 반려견/반려묘 보호자가 자주 소진되는 소모품을 필요할 때 단건으로 구매하고, 반복적으로 필요한 SKU는 정기 구독으로 안정적으로 받아보도록 돕는 커머스 서비스다.

## 해결하려는 사용자 문제

반려동물 소모품은 사료, 간식, 배변용품, 위생용품처럼 사용 주기가 반복되고 품절 또는 구매 지연이 생활 불편으로 바로 이어지는 상품군이다. 사용자는 매번 재고를 확인하고 같은 상품을 다시 찾고 주문하는 과정을 반복해야 하며, 주문을 놓치면 급하게 대체 상품을 찾아야 한다.

PawCycle Commerce의 MVP는 이 반복 부담을 줄이는 데 집중한다. 사용자는 즉시 필요한 상품은 단건으로 구매하고, 예측 가능한 소모품은 정기 구독으로 설정하며, 다음 회차 전에 수량 변경, 건너뛰기, 일시정지, 재개, 해지를 스스로 처리할 수 있어야 한다.

## 핵심 가치

1. 반복 구매 부담 감소: 자주 쓰는 소모품은 사용자가 정한 주기로 다시 주문되도록 한다.
2. 즉시 구매와 정기 구매의 공존: 당장 필요한 상품은 단건 구매로, 꾸준히 필요한 상품은 구독으로 처리한다.
3. 예외 상황의 투명성: 품절, 결제 실패, 판매 중단 같은 상황은 사용자가 이해할 수 있는 이유와 다음 행동으로 표시한다.
4. 가벼운 운영 가능성: 단일 판매자가 상품, 재고, 주문, 구독 현황을 최소한으로 확인하고 조정할 수 있어야 한다.

## 제품 포지션

MVP는 반려동물 전체 케어 플랫폼이 아니라 반려견/반려묘 소모품 재구매와 정기 구독에 특화된 커머스다. 건강 기록, 병원 연계, 맞춤 추천, 커뮤니티, 리뷰, 멤버십 혜택은 제품 차별화의 후보일 수 있지만 PS-001 MVP 범위에는 포함하지 않는다.

## MVP 성공 모습

MVP가 성공한 상태는 사용자가 다음 행동을 완주할 수 있는 상태다.

1. 반려견/반려묘 소모품을 탐색하고 단건 주문을 완료한다.
2. 반복 구매할 SKU를 선택해 2주, 4주, 8주 중 하나의 주기로 구독을 만든다.
3. 다음 회차 전에 수량 변경, 건너뛰기, 일시정지, 재개, 해지를 수행한다.
4. 품절 또는 결제 실패로 이번 회차가 진행되지 않는 이유를 확인한다.
5. 운영자는 상품 판매 가능 여부, 재고, 주문, 구독 현황을 최소한으로 확인한다.

## 설계로 넘기지 않는 것

이 비전 문서는 제품 관점의 입력만 제공한다. 다음 항목은 여기서 확정하지 않는다.

- 엔티티, 값 객체, 애그리거트, 테이블, 컬럼
- 주문, 결제, 구독의 상태 전이와 불변식
- API 경로, 요청/응답, 이벤트, 배치 스케줄
- 프론트엔드 화면 구성, 컴포넌트 구조, 관리자 화면 구현 방식
