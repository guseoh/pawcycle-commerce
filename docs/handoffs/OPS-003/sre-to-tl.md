# OPS-003 SRE → TL 인수인계

## 전달 목적

루트 `AGENTS.md`에 PR 리뷰 한국어 작성 지침을 추가했음을 Tech Lead에게 전달한다.

## 변경 요약

- 모든 GitHub Pull Request 리뷰 댓글은 한국어 작성이 기본이다.
- 코드 식별자, 파일 경로, 요구사항 ID, API path, 커밋 SHA는 원문을 유지한다.
- 심각도 표기 `P0`, `P1`, `P2`는 유지한다.
- 자동 리뷰 기본 문구도 가능한 범위에서 한국어로 작성한다.

## 후속 작업 영향

- PS-003 이후 PR 리뷰에서도 한국어 리뷰 지침을 따른다.
- 리뷰 도구나 GitHub Actions 자동화 자체는 변경하지 않았다.

## 제한

- 영어 리뷰를 자동 차단하는 CI는 추가하지 않았다.
- 리뷰 봇 설정은 변경하지 않았다.
